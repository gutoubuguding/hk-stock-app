## 技术选型

### 整体架构
- **后端：** Java Spring Boot（主力，负责业务逻辑、接口、定时任务、数据持久化）
- **AI微服务：** Python FastAPI（负责新闻AI分析、新股走势预测等AI功能）
- **前端：** Vue.js
- **数据库：** PostgreSQL

### 数据来源
- **股票行情/K线：** Futu OpenAPI
- **新股列表及基本面：** AKShare（港股行情接口）
- **新股详细数据（中签率、超购倍数、基石投资者等）：** 爬虫抓取
- **新闻数据：** 东方财富、新浪财经、港交所披露易、财联社、经济通、雪球、微博
- **AI分析：** Python FastAPI + LLM API

### 爬虫数据源
- **经济通 (etnet.com.hk)** — 港股新股数据全面
- **阿斯达克 (aastocks.com)** — IPO 专区
- **港交所披露易** — 官方公告

### 关键技术栈
- **后端框架：** Spring Boot
- **定时任务：** Spring Scheduler / Quartz
- **数据持久化：** MyBatis / JPA
- **AI微服务：** Python FastAPI + LLM API（可配置多个模型，用户自选）
- **HTTP通信：** Java 调用 Python 微服务（REST API）
- **前端框架：** Vue.js
- **前端图表：** ECharts（K线图）
- **数据库：** PostgreSQL

### 通知方式
- 网页弹窗提醒（新股关键节点、价格预警触发）

### 数据更新机制
- 定时任务（Spring Scheduler）每天1-2次检查更新

## 需求1：港股K线图查询与搜索

### 功能描述
- 支持查询所有港股K线图
- K线周期：日K、5日K、月K、年K
- 搜索功能：按股票代码、名称、类别/板块搜索
- 搜索结果展示列表供用户选择
- 点击后展示对应K线图

### 每只股票当日关键信息
- 开盘价
- 收盘价
- 最高价
- 最低价
- 成交量
- 成交额
- 涨跌幅
- 换手率

---

## 需求2：新闻AI分析

### 功能描述
- 针对指定股票，自动抓取并整理影响股价涨跌的重要新闻
- AI对新闻进行智能分析，输出利空/利好判断

### 新闻来源
- 港股资讯网站（东方财富、新浪财经港股）
- 公司公告（港交所披露易）
- 财经媒体（财联社、经济通）
- 社交媒体舆情（雪球、微博）
- 要求：来源真实可信

### AI总结方式
- 列出关键新闻 + 简要分析利空/利好
- 给出综合判断（如"近期受XX事件影响，偏利空/利好"）

### 时间范围
- 默认展示近1周的新闻

## 需求3：新股AI分析

### 功能一：即将上市新股
- 获取即将上市的新股列表
- AI分析新闻，推断上市后走势（利好/利空/综合判断）

### 功能二：近一年上市新股横向对比表格

#### 表格数据维度
- 股票代码/名称
- 上市日期
- 发行价 vs 现价（当前涨跌幅）
- 上市首日涨跌幅
- 上市后7天涨跌幅
- 上市后30天涨跌幅
- 认购中签率
- 公开发售倍数（超购倍数）
- 公开发售 vs 国际配售比例
- 每手入场费
- 募资金额
- 主营业务/行业板块
- 保荐人/承销商
- 基石投资者及投资金额
- 发行市盈率 vs 同行业平均PE
- 是否纳入港股通（标识）

#### IPO全流程时间线
- 招股期（起止日期）
- 定价日
- 公布中签日
- 上市日
- 支持提前提醒用户关键时间节点（网页弹窗）

#### 数据更新机制
- 定时任务每天1-2次从 Futu OpenAPI 拉取最新IPO列表
- 对比数据库，发现新数据自动更新表格
- 关键时间节点（招股期开始、定价、中签公布、上市）触发网页弹窗提醒

#### 新股板块统计
- 按行业板块统计近一年新股表现（首日涨幅、7天涨幅等）
- 辅助判断哪类新股更值得打

#### 破发率统计
- 近一年新股整体破发比例
- 辅助评估打新风险

#### 表格交互
- 默认按上市时间排序
- 支持按任意维度排序
- 后续新上市股票自动追加到表格中
